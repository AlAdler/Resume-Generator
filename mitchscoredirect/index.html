This creates a super slick and simple resume based off Business Insider's "Excellent Resume". To see an example of a resume this generates, click here.

Here are a few things in mind when using this app:

  • This has been tested on Chrome only, but should work on Firefox as well.
  • Pay attention to spelling and capitalization. I don't fix that for you.
  • Refreshing the page will clear everything if you mess up.
  • So you don't lose any data, it would be best to type everything into a word document on your computer, then paste it in here.
  • When you're saving your resume, you can either print it directly, or change the destination to save as a PDF. Detailed instructions here.
  • If you know how, you can format the textareas using HTML tags.
  • No data you enter here is stored or sent anywhere. I promise!
